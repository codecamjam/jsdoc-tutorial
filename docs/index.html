jsdoc-tutorial

This is a code-along with Brad Traversy's JSdoc tutorial video on youtube.

https://www.youtube.com/watch?v=YK-GurROGIg

Instructions for setting this jsdoc tutorial:

  1. npm init -y
  2. npm i -D jsdoc
  3. touch jsdoc.json
  4. mkdir src (for putting all your js files)
  5. npm run doc
  6. Rinse, repeat
